安装到手机桌面

它本质上是一个网页，但可以像 App 一样从手机桌面打开。

iPhone / iPad

  1. 用 Safari 打开电子书链接。
  2. 点击底部或顶部的“分享”按钮。
  3. 选择“添加到主屏幕”。
  4. 确认名称，例如“赢在执行”。

注意：微信内置浏览器通常不支持完整安装，请用 Safari 打开。

安卓手机

  1. 用 Chrome 或 Edge 打开电子书链接。
  2. 如出现提示，点击“安装应用”。
  3. 如果没有提示，点右上角菜单。
  4. 选择“安装应用”“添加到主屏幕”或“添加到手机”。

如果菜单里没有这些选项，请复制链接到 Chrome 或 Edge 打开；微信、企业微信和部分国产浏览器可能不提供 PWA 安装入口。Edge 不一定主动弹出提示，可通过菜单手动添加。

首次打开需要联网。应用只缓存基础界面，电子书页面在阅读时按需加载，避免手机浏览器一次性缓存过多资源导致卡顿。
